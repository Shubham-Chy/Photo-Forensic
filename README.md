## üöÄ ESTABLISH NODE (Installation)

1. **Prerequisites**: Node.js 18.x and a Gemini API Key.
2. **Deploy**:
   ```bash
   git clone https://github.com/shubham-chy/Photo-Forensic.git
   cd Photo-Forensic
   npm install
   ```
3. **Configure**: Create `.env` from `.env_example` and insert your key.
4. **Launch**: `npm run dev`
